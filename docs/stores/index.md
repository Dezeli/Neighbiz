# 🏪 Stores API

`/api/v1/stores/` 하위에서 제공되는 가게 관련 API입니다. 사용자 본인의 가게 등록, 가게 정보 조회, 마이페이지 조회 기능을 포함합니다.

| 기능                       | 메서드  | 엔드포인트                    |
| ------------------------ | ---- | ------------------------ |
| [가게 등록](create.md)       | POST | `/api/v1/stores/`        |
| [내 가게 정보 조회](me.md)      | GET  | `/api/v1/stores/me/`     |
| [마이페이지 정보 조회](mypage.md) | GET  | `/api/v1/stores/mypage/` |
