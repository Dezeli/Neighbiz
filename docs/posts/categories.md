# 📂 카테고리 목록 조회 API

## 📥 GET `/posts/categories/`

게시글 작성 시 선택할 수 있는 제휴 카테고리 목록을 조회합니다.

### 🔹 Response 200 (성공)

```json
{
  "success": true,
  "message": "카테고리 목록을 불러왔습니다.",
  "data": [
    {"id": 1, "name": "카페"},
    {"id": 2, "name": "음식점"},
    {"id": 3, "name": "기타"}
  ]
}
```

### 🔖 설명

* 카테고리는 서버에서 미리 정의되어 있으며, 클라이언트는 이 목록을 통해 선택지를 구성합니다.
* 게시글 작성 시 이 ID 값 배열을 함께 전송해야 합니다.
